
## Setup xdebug in phpStorm
![phpStorm xdebug config screenshot](./img/xdebug.png)
* Setup path mappings.
    - In preferences, find PHP servers.
    - Add appserver.
    - Map `/wordpress/index.php` to `/app/wordpress/index.php`
    - Map `/wordpress/index.php` to `/app/wordpress/index.php`
* Turn on debug listening -- phone icon.
